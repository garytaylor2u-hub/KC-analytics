# Microsoft Copilot for Analysts

Real-world examples of using **Microsoft 365 Copilot** in Excel and PowerPoint to turn raw data into insights and presentations.

### What It Shows
- Upload CSV → Copilot instantly analyzes trends, anomalies, and gaps
- Ask natural language questions → get charts, summaries, and slide decks
- Zero code — 100% native M365 Copilot (2025)

### Files
- `sales_with_forecast.xlsx` → Original data
- `sales_trends.pptx` → Fully AI-generated deck (slides, charts, insights)
- Screenshots → Before/after Copilot transformation

Built & documented November 2025
